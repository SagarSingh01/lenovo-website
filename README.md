# 💻 Lenovo Website Clone

A responsive **Lenovo-like website** built using **HTML, CSS, and JavaScript**.  
This project replicates the look and feel of Lenovo's official website, including product sections, navigation, sliders, and responsive layout.

---

## 🚀 Features

- 🖥️ Homepage with featured products and banners  
- 📱 Fully responsive design for mobile and desktop  
- 🔄 Image sliders and product carousels  
- 🛒 Product sections with hover effects  
- 📜 Footer with links, social icons, and contact information  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox, Grid, Animations)  
- JavaScript (DOM manipulation, sliders, interactivity)
